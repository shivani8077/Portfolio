Spam Mail Classification App

A machine learning-powered web app that detects spam emails in real time. Built using Python and Streamlit, the app uses NLP techniques like tokenization, stopword removal, and stemming, with features extracted using TF-IDF. A Multinomial Naive Bayes model classifies emails with 97.7% accuracy and 1.0 precision. The project includes an interactive UI for users to input messages, view prediction results, and explore visual insights. Developed during the TechSaksham–AICTE Internship, it demonstrates the complete ML pipeline from preprocessing to deployment.
Tech Stack: Python, Streamlit, scikit-learn, NLTK, pandas, TF-IDF, matplotlib
